# AdminApp_GQL_ExpressJS
AdminApp Backend based on GQL and ExpressJS

---
## Prerequirements

1. Clone the repository by using `git clone <repository>` to any place on your local machine.
2. Change into the cloned file cloned by using `git clone <repository>` through `cd <path/to/your/repository/on/your/local/machine>` 
3. Install all dependencies used by the project `npm install`
4. Create a `.env` file for database configuration with following fields:

```
DB_TYPE=
DB_USER=
DB_PASSWORD=
APP_PORT=
```

## Development

1. For development use the command `npm run dev`
